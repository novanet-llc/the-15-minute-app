# The 15-minute app

An app for tracking your time and activities in 15-minute slots. The goal is to find the "time leaks" on your day, which prevents you to maximise your productive output cumulated over the weeks.

If you are not familiar with the 15-minute theory, please check the [15-minute rule](https://www.youtube.com/shorts/om_6WSfRLZ8) YouTube video by Alex Hormozi.

# An Expo app

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
